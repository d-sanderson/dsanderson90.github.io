RS21 Technical/Menu

Navigation

Click on the ? button to show the sidebar again when you've closed it.

Click on the < button to return to the default map view.

Click on the overlays button directly above the home view button to toggle map layers.

Click on the blue facebook markers to see the details for the selected Facebook Place

Hover over the light-blue twitter markers to see user info

Click anywhere on the map to close the sidebar


Filtering

Filter Facebook Places

Select a place category to see all FB Places within that category.

Filter Tweets

Select a user to see their tweet(s), location(s), and time(s) of tweet.

Filter Census Bureau Data

Select a census code
