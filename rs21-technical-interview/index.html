Navigation

Click on the ? button to open/close the sidebar.

Click on the < button to return to the default map view.

Click on the overlays button directly above the home view button to toggle map layers.

Hover over the blue facebook markers to see the details for the selected Facebook Place.

Click on the light-blue twitter markers to see user info.

Click anywhere on the map to close this sidebar.


Filtering

Filter Facebook Places

Q and A from Prompt

Why you chose the strategy you used?

My strategy for this project was to implement all of the datasets and create a UI that is both intuitive and represents the data in a beautiful way. My reasoning for this approach was that if everything could be filtered, then the data could be examined in a variety of novel ways.

What you found most interesting about the project?

The most interesting part of this project was playing with the ACS Data and figuring out how to dynamically visualize it on the map.
